# 每日热点新闻摘要

自动搜索并整理当日热点新闻，生成美观、结构化的新闻摘要报告。

## MCP 服务配置

本技能依赖以下 MCP 服务，运行时**必须**先读取 `mcp-config.json` 获取服务 URL：


| 环境变量名               | MCP 服务 | mcpId |
| ------------------- | ------ | ----- |
| `$BOCHA_SEARCH_URL` | 博查搜索   | 1039  |


### 配置读取方式

**约束：调用方 agent 必须在执行任何 MCP 调用前，先读取 mcp-config.json 文件**

```python
import json

with open("mcp-config.json", "r", encoding="utf-8") as f:
    config = json.load(f)

# 获取博查搜索服务的 URL
bocha_url = config["BOCHA_SEARCH_URL"]["url"]
```

## 工作流程

1. **读取配置**：首先读取 `mcp-config.json` 获取博查搜索服务的 URL
2. **搜索热点**：调用博查搜索获取当日热点新闻
  - 使用关键词："今日热点新闻"、"今日要闻"、"最新资讯"
  - 每个关键词搜索获取 5-8 条结果
3. **智能分类**：将新闻按领域分类（科技、财经、社会、国际、娱乐等）
4. **生成摘要**：为每条新闻生成简洁的摘要（50-80 字）
5. **美化输出**：按规范格式输出结构化新闻摘要

## 新闻搜索策略

执行多次搜索以覆盖不同领域：

```bash
# 综合热点
python3 scripts/call_mcp.py call "$BOCHA_SEARCH_URL" web_search --params '{"query": "今日热点新闻", "count": 8}'

# 科技资讯
python3 scripts/call_mcp.py call "$BOCHA_SEARCH_URL" web_search --params '{"query": "今日科技新闻 最新", "count": 5}'

# 财经动态
python3 scripts/call_mcp.py call "$BOCHA_SEARCH_URL" web_search --params '{"query": "今日财经新闻 股市", "count": 5}'

# 国际要闻
python3 scripts/call_mcp.py call "$BOCHA_SEARCH_URL" web_search --params '{"query": "国际新闻 今日要闻", "count": 5}'
```

## 输出格式规范

```markdown
> ⏰ **生成时间**：{HH:MM}

[查看原文]({原文链接})

*📢 本摘要由 AI 自动生成，仅供参考*
```

### 格式要求

1. **标题层级**：使用 `#` 作为主标题，`##` 作为分类标题，`###` 作为新闻条目标题
2. **视觉元素**：
  - 使用 emoji 图标增强可读性（📰🔥💻💰🌍📌🕐）
  - 使用 `---` 分隔线区分不同新闻条目
  - 使用引用块 `>` 展示元信息
  - 每一条热点新闻后面，要带上查看原文的链接
3. **内容规范**：
  - 摘要控制在 50-80 字，突出重点
  - 保留原文链接，方便用户深入了解
  - 标注信息来源和发布时间
4. **分类标签**：
  - 🔥 今日热点：综合类重要新闻
  - 📌 AI热点：AI相关的最新动态
  - 💻 科技资讯：科技、互联网、AI 相关
  - 💰 财经动态：股市、经济、商业相关
  - 🌍 国际要闻：国际政治、外交相关
  - 🎬 娱乐体育：娱乐、体育相关（如有）

## 使用示例

**用户输入**：

- "今天有什么热点新闻？"
- "帮我整理一下今日要闻"
- "每日新闻摘要"
- "最近发生了什么大事？"

**技能执行**：

1. 读取 `mcp-config.json` 获取博查搜索 URL
2. 调用博查搜索获取各领域新闻
3. 智能分类并生成摘要
4. 按规范格式输出美观的新闻摘要

## 注意事项

1. 每次搜索获取的新闻要去重，避免重复报道
2. 优先选择权威媒体来源的新闻
3. 摘要要客观中立，不添加主观评价
4. 如某分类无相关新闻，可省略该分类章节
5. 确保所有外部链接可正常访问
